# Consulta de Base de Datos con Streamlit

Proyecto para consultar una base de datos en modo de solo lectura utilizando Streamlit.

## Tecnologías

- Python
- Streamlit
- Pandas

## Estado

🚧 En desarrollo

## Recordar

utilizar las siguientes lineas de codigo en el codespace para instalar el *Streamlit*

cd /workspaces/Consulta-Online-Base-Limpia

python3 -m pip install --upgrade pip

python3 -m pip install streamlit pandas openpyxl

python3 -m streamlit run app.py
